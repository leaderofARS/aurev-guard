# AurevGuard Backend

The backend API for AurevGuard, built with Node.js and Express. It serves as the bridge between the frontend dashboard and the AI model's results.

## Features

- **API Endpoints**: Serves wallet risk scores and transaction data.
- **Orchestration**: Triggers the Python AI pipeline (optional integration).
- **Database**: (If applicable) Stores user data and cached results.

## Setup

```bash
npm install
npm run dev
```

# Source Code

This directory contains the Python source code for the AI model.

## Modules

- **`live_pipeline.py`**: Entry point for real-time data fetching and processing.
- **`feature_engineering.py`**: Transforms raw transaction data into ML-ready features.
- **`agent.py`**: Defines the autonomous agent logic.
- **`graph_features.py`**: Extracts graph-based metrics (centrality, degree, etc.).
- **`inference.py`**: Runs the trained models on new data.
- **`train.py`**: Script for training the risk assessment models.
- **`exporter.py`**: Utilities for exporting results.
